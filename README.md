# SMI · Statistical Modelling and Inference

统计建模与推断（Statistical Modelling and Inference）课程的学习资料与复习笔记归档。

## 目录

- `notes/`：复习笔记、知识梳理和样题解析的 LaTeX 源文件及 PDF。
- `course-materials/slides/`：课程课件。
- `course-materials/exams/`：样卷、历年卷和参考答案。
- `course-materials/reference/`：课程参考资料。
- `course-materials/archive/2025/`：2025 届课程资料，包括作业、实践和参考资料。
- `assignments/`：本课程作业材料。
- `practice/`：练习题及其源码、图片。

## 使用说明

LaTeX 笔记的源文件与对应 PDF 一并保留，便于阅读、复现和修订。R、R Markdown 和数据文件保留在其原有资料目录中，以维护相对路径依赖。
